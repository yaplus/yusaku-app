
# Yusaku-App（仮）

https://yusaku.herokuapp.com/

## 「昔◯◯してたアイツ、生きてんのかな」を見れる形にしよう。

Yusaku-App（仮）。昔の知人のユニークなエピソードを、匿名で気軽に。  
昔の知人のエピソードを紹介することで、エピソードを知っている人がその知人の生存を教えてくれる、革新的で魔法のようなサービスです。
